extensional

An occurrence of a term that has an extension is extensional iff the replacement of that term with another that has the same extension (i.e. is true of the same tuples of objects) produces a sentence whose truth value is the same as the original.

caveat - this restricts to only terms with extensions. It seems straightforward to extend this to include truth functional connectives
