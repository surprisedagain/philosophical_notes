I agree with Russell's claim in On the Notion of Cause that causation is an obsolete notion that can be dropped from our theories of the world. In modern jargon this appears to be what David Lewis means by a Humean view of causation.

Russell's argument from contiguity

In On the Notion of Cause Russell claims that causes must be temporally contiguous with their effects. For if they are not, then there is some interval between causes and effects and However short we make the interval, something may happen during this interval which prevents the expected result.

Hence the cause in question does not necessitate its effects and is therefore not the 'total' cause (i.e. it does not completely determine the effect). But if time is continuous then no two moments in time are contiguous. For any two instants in the same continuous time stream (in special relativity this would be a set of space-time points that are simultaneous from some inertial reference frame) there is a determinate time interval between them. If that interval is 0 seconds then the two instants are identical. If the interval between them is not 0 seconds then, since time is continuous, it contains an infinite number of instants, and hence the original two instants are not contiguous.

If we add that no causal influence can propagate faster than a fixed speed (such as the speed of light) then action at a distance is prohibited, and much the same argument applies to spatial contiguity.

Of course time intervals may still be contiguous, provided one is an open and the other a closed set of time instants (at the boundary between them). For instance the set of instants t such that 1:59am≤t≤2am and the set of instants such that 2am<t≤2:01am are contiguous.
