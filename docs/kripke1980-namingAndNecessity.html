weirdness with non-rigid designators

Lets focus on the last sentence:

If 'a' and 'b' are not rigid designators, no such conclusion follows about the statement 'a = b' (though the objects designated by 'a' and 'b' will be necessarily identical).

So lets say that Ali is the only red car on my street. Then Ali rigidly designates Ali and the definite description the red car on my street non-rigidly designates Ali. So too if Ali is the only Holden on my street then the definite description the Holden on my street also non-rigidly designates Ali.

Now it is not true that Necessarily the red car on my street is the Holden on my street but it is true that Necessarily, the object designated by 'the red car on my street' is identical to the object designated by 'the Holden on my street'. So within the scope of a □ replacing a name N with the object referred to be 'N' does not always preserve truth.

More generally □ P(the object referred to by 'Name') may be true and yet □ P(Name) be false. (We don't have to consider P as just a predicate here, but as any closed formula.) It is, I suppose, true that Necessarily the object designated by 'the red car on my street' was built in China. but false that Necessarily the red car on my street was built in China. but

We have not quite broken the principle that substituting any name for both instances of 'N' in 'N' designates N produces a truth. (A principle that seems seems every bit as non-negotiable as 'P' is true iff P.) But we do have a startling failure of substitutivity that comes close.

Naively one would have thought that quotation marks around a term (name or definite description) and the object refered to by …, would express kind of inverse operations. Just as quotation marks applied to sentences and is true. When one applies is true to a quoted sentence one gets another that is true just in case the quoted original is true. Indeed this is what (I thought) the disquotational theory of truth was all about. (See disquotational_truth)

So a sentence containing Necessarily can go from false to true by replacing the red car on my street with the object designated by 'the red car on my street'

This seems quite a high price to pay. For non rigid designators within the scope of necessity operators substitutivity fails.
